# Script Instructions — pointer

The full rules live in `shared-context/workshop-rules.md`. Read that file at the start of any workshop. It covers:

- Voice (co-learner, mirror-then-redirect, micro-praise, banned/preferred phrases)
- The two modes (coach mode + work mode)
- Decision points (always offer "or let me pick")
- Pattern-finding (Claude finds, learner reacts)
- Re-anchor every 3-4 beats
- Heavy data work = script (fast vs. slow)
- Just-in-time tool tips
- Script markers (`Say:`, `Suggest something like:`, `Decision:`, `Check:`, etc.)
- Stay in character / no fourth wall
- Logging format (`workshop-log.md`)
- FX reference (1 USD = 32 THB + table)
- File extensions (.md > .txt for Cursor)
- Slash command shape
- IDE basics teaching distribution
- Cursor compatibility

**For workshop authors:** when designing a new workshop, see the design repo's `knowledge/workshop-story-builder-v2.md`.
